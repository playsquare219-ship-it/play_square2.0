# Play Square - تطبيق إدارة فرق كرة القدم

تطبيق ويب حديث لإدارة فرق كرة القدم في الجزائر

## المميزات

- ✅ نظام تسجيل دخول وإنشاء حساب آمن
- ✅ إدارة الفرق واللاعبين
- ✅ متابعة المباريات والنتائج
- ✅ واجهة مستخدم عصرية وسهلة الاستخدام
- ✅ دعم كامل للغة العربية

## التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم
- **Radix UI** - مكونات واجهة المستخدم
- **Firebase Firestore** - قاعدة البيانات السحابية
- **Firebase Authentication** - نظام المصادقة الآمن

## التشغيل المحلي

```bash
# تثبيت المكتبات
npm install

# تشغيل المشروع
npm run dev
```

المشروع سيعمل على: `http://localhost:3000`

## نظام المصادقة والبيانات

يستخدم التطبيق:
- **Firebase Authentication** - للمصادقة الآمنة (Email/Password و Google Sign-In)
- **Firebase Firestore** - لتخزين جميع البيانات (المستخدمين، الفرق، المباريات، إلخ)
- **JWT Tokens** - لإدارة الجلسات
- **bcryptjs** - لتشفير كلمات المرور

## مباريات My Matches
- تبويب **My Matches** يعرض فقط المباريات التي يشارك فيها فريق المستخدم المسجل.
- يتم تحديد المباريات الخاصة بالمستخدم من خلال `team1Id` أو `team2Id` المرتبط بفريقه.
- يتم التحقق من الجلسة عبر الكوكي الآمن `playSquareToken` قبل جلب مباريات الفريق.
- يمكن إلغاء المباراة عبر `PATCH /api/matches` باستخدام `matchId` و`action: cancel`.
